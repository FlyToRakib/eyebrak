# Eyebrak

**Smart breaks, Pomodoro cycles, eye care tips, and streak tracking for digital wellness.**

Eyebrak helps you stay productive and healthy with:

- Smart break reminders
- Pomodoro-style focus cycles
- Eye care tips
- Streak tracking to encourage consistent breaks

## Installation

1. Download the extension from this repository or Chrome Web Store.
2. In Chrome, go to `chrome://extensions/`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the `eyebrak` folder.
